## Summary

宝宝成长记录语音版是一款面向宝妈的H5应用，通过语音输入+AI智能识别，帮助记录宝宝日常活动（喂奶、换尿布、大便、辅食、睡眠）。应用采用Vue 3 + Vite构建前端，NestJS + SQLite构建后端，实现可爱的治愈系UI风格。

核心技术特点：
- 长按语音输入，AI自动识别记录类型和详情
- 智能时间解析（支持"刚才"、"昨天"等模糊表达）
- 补充对话限制在两轮内，保持简洁体验
- 数据可视化统计（奶量、尿布、大便、睡眠趋势）
- 用户注册登录，密码加密存储

## Technology Stack Summary

### Frontend
- **Framework**: Vue 3 with Composition API
- **Build Tool**: Vite 5.x
- **State Management**: Pinia
- **HTTP Client**: Axios with interceptors
- **UI Components**: Element Plus / Naive UI (cute theme)
- **Charts**: ECharts
- **Date Handling**: Day.js
- **Testing**: Vitest

### Backend
- **Framework**: NestJS 10.x
- **Language**: TypeScript 5.x
- **Database**: SQLite 3 with TypeORM
- **Authentication**: JWT (access + refresh tokens)
- **Password Hashing**: bcrypt
- **AI Integration**: OpenAI API (Whisper + GPT-4)
- **Validation**: class-validator
- **Testing**: Jest

### Security Measures
- JWT authentication with short-lived access tokens
- bcrypt password hashing (cost factor 12)
- CORS whitelist
- Rate limiting (100 req/min general, 10 auth req/min)
- Input validation and sanitization
- SQL injection prevention via TypeORM
- HTTPS in production

### Performance Optimizations
- Lazy route loading
- API response caching (stats)
- Database indexing
- Gzip/Brotli compression
- Virtual scrolling for long lists
- Debounced voice recording
